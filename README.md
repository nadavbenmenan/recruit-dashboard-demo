# דשבורד ניהול גיוס — הדגמה סטטית

לוח בקרה אינטראקטיבי לניהול הליך מיון וגיוס, במצב **הדגמה חיה** על **נתונים
סינתטיים לחלוטין**. האתר רץ כולו בדפדפן (HTML/CSS/JS), ללא שרת וללא מסד נתונים.

**אתר חי:** https://nadavbenmenan.github.io/recruit-dashboard-demo/

## מה יש בפנים
- **דוח בקרה מנהל** — מצב ההליך כרגע, 5 טווחי סטטוס, ונורות SLA לפי יחידה.
- **דשבורד דינאמי** — סינון חי לפי תפקיד · יחידה · ימי המתנה · שלב בהליך, עם מתג
  אוכלוסייה (בהליך / כולם) וטווח תאריכים.
- **צפי להליך המיון** — חיזוי 4 שבועות (תוחלת, לא נתון).
- **מסירי מועמדות · הפסקות הליך** — פילוח אירועים לפי שלב, יחידה, תפקיד וסיבה.

## הדגמה — קריאה בלבד
- כל הנתונים **מומצאים** ואינם משקפים נתוני אמת. 60,000 מועמדים, 6,000 בהליך.
- אין טעינה, עריכה או ייצוא — כפתורי הכתיבה מנוטרלים.
- הנתונים נוצרים ע"י `gen_data.py` (random עם seed קבוע) ונשמרים ב-`data/*.json`.

## הרצה מקומית
```bash
python3 gen_data.py          # מייצר data/*.json
python3 -m http.server 8000  # ואז פתח http://127.0.0.1:8000/
```

## מבנה
- `index.html` · `style.css` — המסך (RTL).
- `app.js` — מנוע חישוב צד-לקוח שמשחזר את שכבת ה-API מעל הנתונים הסינתטיים.
- `data/` — הנתונים הסינתטיים (candidates / events / meta).
- `gen_data.py` — מחולל הנתונים.
